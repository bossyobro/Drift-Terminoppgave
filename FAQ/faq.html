Frequently Asked Questions

1. What is the objective of this project?

The objective of this project is to create the infrastructure for a small to medium-sized company.

2. What operating system is being used?

We are using Windows Server 2022 for the server infrastructure.

3. What is OPNSense?

OPNSense is an open-source firewall and routing platform that will be used as a virtual appliance in our infrastructure.

4. How is the network configured?

The network is configured using VLANs and includes a primary server, a backup server, and a firewall.

5. How are users managed in the system?

Users are managed through Active Directory, where we create Organizational Units (OUs) to organize users and resources.
